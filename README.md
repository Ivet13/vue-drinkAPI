# drinkAPI

#### API to show random drink drink, searches for specific drink and shows the recipe of the drink
#### Contains also login mockup

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
